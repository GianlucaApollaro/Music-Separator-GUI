===============================================================================
                          MUSIC SEPARATOR - MANUALE UTENTE
===============================================================================

Benvenuto in Music separator, uno strumento avanzato di separazione audio basato 
sull'Intelligenza Artificiale (architetture Roformer e MDX). Questa 
applicazione ti permette di isolare voci, strumenti e altri elementi con 
una qualità da studio.

1. INSTALLAZIONE
-------------------------------------------------------------------------------
WINDOWS:
- Esegui 'install.bat' per creare l'ambiente e installare le dipendenze.
- Usa 'run.bat' per lanciare l'applicazione.
- Se non hai una GPU NVIDIA, usa 'install_cpu.bat' e 'runCpu.bat'.

macOS (Apple Silicon M1/M2/M3/M4):
- Assicurati di avere Homebrew installato (brew.sh).
- Installa le dipendenze via Terminale: brew install ffmpeg libsndfile python@3.12
- Fai tasto destro -> 'Apri' sul file 'install_mac.command'.
- Usa 'run_mac.command' per lanciare l'app.
- Leggi README.txt nella cartella Mac per risolvere problemi di permessi (chmod +x).

2. UTILIZZO BASE (SINGOLO MODELLO)
-------------------------------------------------------------------------------
1. Seleziona il file audio d'ingresso (WAV, FLAC, MP3, ecc.).
2. Scegli una Cartella di Output.
3. Seleziona un Modello dal menu a tendina (divisi per tipo: Vocali, Batteria, ecc.).
4. Scegli la profondità di Bit (16-bit standard, 32-bit per alta fedeltà).
5. Clicca "Avvia Separazione".

3. ENSEMBLE NATIVO (FUSIONE MODELLI)
-------------------------------------------------------------------------------
Spunta la casella "Ensemble" per selezionare DUE modelli diversi. L'applicazione 
elaborerà il file con entrambi e fonderà matematicamente i risultati per 
ottenere una separazione superiore.

4. PRESET ENSEMBLE A CATENA (AVANZATO)
-------------------------------------------------------------------------------
Il menu "Preset" permette di eseguire processi "A Catena" dove l'output di 
un modello viene passato automaticamente a un secondo modello.

*Nota: Selezionare un Preset annulla la selezione del singolo modello.*

PRESET DISPONIBILI:
- Separazione Vocali: Separa Voce/Base, poi divide la Voce in Lead e Backing.
- De-Reverb Vocali: Isola la Voce, poi rimuove riverbero ed eco dal cantato.
- Ultimate Stem (7 Tracce): Estrae la Voce e la Base ad alta qualità, 
  poi divide la base in 5 parti (Basso, Batteria, Piano, Chitarra, Altro).
- Separazione Uomo/Donna: Isola specificamente voci maschili vs. femminili.
- Restauro Live: Estrae il pubblico (Applausi), poi pulisce il palco 
  separandolo in Voce e Base strumentale.

5. MODELLI SPECIALIZZATI
-------------------------------------------------------------------------------
- Drum Separation: Divisione della batteria in Kick, Snare, Toms e Piatti.
- Aspiration: Isola i respiri e le prese di "fiato" vocale per pulizie chirurgiche.
- Crowd: Perfetto per pulire registrazioni Live dai rumori del pubblico.

6. NOMI DEI FILE DI OUTPUT
-------------------------------------------------------------------------------
Il programma usa suffissi chiari per una facile identificazione:
_Vocals, _Instrumental, _Lead, _Backing, _DeReverb, _Male, _Female, _Crowd.

7. ACCELERAZIONE HARDWARE
-------------------------------------------------------------------------------
- Windows: Utilizza CUDA (NVIDIA) o la CPU.
- macOS: Utilizza MPS (Metal Performance Shaders) sui chip serie M per 
  un'elaborazione velocissima tramite il Neural Engine.
