1. Installation

Windows (CPU or GPU)

If you downloaded the standalone precompiled version:

  1. Download the archive format .7z suitable for your machine (GPU version for NVIDIA CUDA graphics cards, or CPU version for other cards or processors).
  2. Extract the archive using a tool like 7-Zip.
  3. Launch the application directly by clicking on Music separator.exe (GPU) or Music separator CPU.exe (CPU).

If you are installing from source code:

  • Run install.bat (or install_cpu.bat) to create the Python virtual environment and download dependencies.
  • Run run.bat (or run_Cpu.bat) to launch the program.

macOS (Apple Silicon M1/M2/M3/M4)

  1. Download the Mac package in .zip format.
  2. Extract the archive and move the Music separator.app folder to your /Applications folder.
  3. Right-click on the application and select Open (only required for the first launch to bypass Apple Gatekeeper's security checks).

Note: FFmpeg is included in the package. If you encounter permission blocks, open your Terminal and run:

xattr -cr "/Applications/Music separator.app"

2. What's New in v1.5 New

  • Enable Preview: A convenient checkbox that, when ticked, cuts and separates a 30-second portion of the audio (you can choose between the first 30 seconds or the final 30 seconds via the dropdown selector that appears next to it). The generated stems are saved in a dedicated folder named previews at the same level as your selected output folder. Great for quickly testing model quality on your tracks.
  • Bandit & SCNet Support: Added full support for loading and separating audio with the new state-of-the-art Bandit v2, Bandit Plus, and SCNet architectures directly in the GUI. Features automated model source code fetching and lightweight runtime mocking to keep dependencies clean.
  • Only Drums + Backing Preset: Introduced the option to mix all non-selected stems into a single output file for single models. This enabled the new preset_drums_no_drums preset, which isolates _Drums and merges everything else into _No_Drums.
  • Integrated Auto-Updater: Music separator now includes an automatic update system. At application startup (or manually via Help -> Check for updates...), the program checks for a new release on GitHub. If available, it downloads and installs it in one click without altering your settings or logs.
  • Automatic Pre-Processing:
    • Normalization: Input audio is automatically normalized to a -0.1 dBFS peak before processing to maximize the AI models' output quality.
    • Stereo Downmix: Multi-channel files (e.g. 5.1, 7.1) are automatically downmixed to stereo to prevent AI model crashes.

3. Basic Usage (Single Model)

To start a classic separation with a single model:

  1. Drag and drop or browse your input file in the Input Audio File field.
  2. Select an Output Directory.
  3. Click on the Model field to open the tree picker and select a model (categorized by logical groups). You can use the built-in search bar at the top of the picker to filter and quickly find models by typing part of their name.
  4. Select the output format (WAV, FLAC, or MP3 at 320kbps).
  5. Click on Start Separation.

4. Native Ensemble (Mixing Two Models)

Check the Enable Ensemble box to combine the outputs of two different models:

  • Choose the primary model (Model 1) and the secondary model (Model 2).
  • Choose the Ensemble Algorithm to blend the waveforms (e.g. avg_wave to average the tracks, min_wave, max_wave, or median_wave).
  • Recommended: The default avg_wave option generally produces the cleanest and most natural stems.

5. Chained Ensemble Presets (Advanced)

The Preset dropdown allows you to run complex multi-model processes where the output of one separation is automatically fed into another model for further cleaning or stem split.

Available chained presets include:

  • Vocal Split: Separates instrumental and vocals, then splits the vocal track into Lead vocals and Backing vocals.
  • Vocal De-Reverb: Extracts vocals and automatically removes background reverb and echo.
  • Ultimate Stems (7 Tracks): Extracts vocals and instrumental, then splits the instrumental track into 5 stems (Bass, Drums, Piano, Guitar, Other) and saves an extra _Extra file.
  • Ultimate Stems + Drum Split: Powered by the new 4-pass engine, it separates Lead vocals, Backing vocals, and splits the instrumental track into Bass, Piano, Guitar, and splits the drums into individual components (Kick, Snare, Cymbals, Toms, etc., for over 10 output files).
  • Guitar Extraction: Preset optimized to isolate lead/rhythm guitars from the rest of the song.
  • Male/Female Split: Splits the vocal track specifically isolating male vs. female voices.
  • Live Restore: Isolates and extracts crowd noise or applause, then divides the music performance into backing and clean vocals.
  • RVC Vocal Prep: Isolates an exceptionally clean Lead vocal track to be used as an RVC training dataset.
  • Only Drums: Extracts the drum kit, removing all melodic or vocal elements.
  • Only Drums + Backing: Extracts the drum kit (_Drums) and merges all vocals, bass, guitar, piano, and other instruments into a single backing track (_No_Drums).

6. Output Options & Cleanup

  • Save in subfolder: If checked, automatically creates a subfolder named after the input song. Recommended for keeping chained/multi-stem outputs organized.
  • Delete silent stems: If the generated stems are quieter than -50 dB (essentially pure silence), the program automatically deletes them to keep folders clean.
  • Play Stem: Click this button after separation completes to open a dialog and quickly play any generated stem in your system's default media player.

7. Chunking (Out-of-Memory Prevention)

If you experience application crashes due to full VRAM (graphics card memory) or if you process very long files (e.g. DJ sets):

  • Check Enable Chunking.
  • Select a duration in minutes (e.g. 2 or 5 minutes). The audio will be split into temporary segments, processed in blocks, and seamlessly merged back together.

8. Batch Processing (Multi-file and Folders)

You can load and process multiple files at the same time:

  • Use the Browse button and select multiple files in the dialog to queue them.
  • Click on Browse Folder to load all supported audio files within a directory (and its subdirectories).
  • The interface will display both the overall progress and the current song progress in real time.

9. Technical Notes & Suffixes

Output files will contain specific suffixes indicating the isolated stem:

  • _Vocals: Combined vocal track.
  • _Instrumental: Complete backing track.
  • _Lead: Main lead voice.
  • _Backing: Backing vocals and choruses.
  • _DeReverb: Dry voice without reverb.
  • _Drums: Drum kit.
  • _No_Drums: Complete backing track excluding the drums.
  • _Bass: Bass guitar / synth bass.
  • _Guitar: Guitar tracks.
  • _Piano: Acoustic / electric piano.
  • _Other / _Extra: Synthesizers or uncatalogued elements.
