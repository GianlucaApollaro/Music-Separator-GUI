Welcome to Music separator, a powerful AI-driven audio separation tool based on 
the state-of-the-art Roformer and MDX architectures. This application allows 
you to isolate vocals, instruments, and specific elements with studio quality.

OFFICIAL DOWNLOAD:
You can always download the latest version of Music separator from:
https://github.com/GianlucaApollaro/Music-Separator-GUI/releases/latest

1. INSTALLATION
-------------------------------------------------------------------------------
WINDOWS:
- Download the version suitable for your system (.7z for CPU or GPU).
- Extract the archive and launch 'Music separator.exe' (GPU version)
  or 'Music separator CPU.exe' (CPU version) directly.
- If installing from source:
  - Run 'install.bat' (or 'install_cpu.bat') to install dependencies.
  - Use 'run.bat' (or 'run_Cpu.bat') to launch the application.

macOS (Apple Silicon M1/M2/M3/M4):
- Download the Mac version (.zip) and extract the archive.
- Move the resulting folder to /Applications.
- Right-click 'Music separator.app' and choose 'Open'.
- NOTE: FFmpeg is included in the package. If you face permission issues, 
  refer to the README for 'xattr' command instructions.

2. WHAT'S NEW in v1.6
-------------------------------------------------------------------------------
- User-Customizable Presets: You can now create your own custom presets by
  clicking the "Create Preset" button next to the preset selector. Supports
  single model and multi-pass chained setups, final output renaming, and
  mixing remaining stems. Custom presets are saved locally inside the program's
  folder for 100% portability.
- Downloaded Models Grouping: Locally downloaded models are automatically
  grouped into a dedicated category ("Downloaded Models (Offline)") at the very
  top of the selector for quick offline access and easy management.
- macOS VoiceOver Accessibility: Refactored model tree selector on macOS
  to use native list controls, making it fully accessible for screen readers
  and VoiceOver.

WHAT'S NEW in v1.5:
- Enable Preview: Cuts and separates a 30-second portion of the audio (first or
  final 30s) in a dedicated folder named 'previews'.
- Bandit & SCNet Support: Dynamic loading and mock-dependencies for ZFTurbo models.
- Auto-Updater: Automatic update checks at startup or via the Help menu.

3. BASIC USAGE (SINGLE MODEL)
-------------------------------------------------------------------------------
1. Select your input file (WAV, FLAC, MP3, MP4, MKV, etc.).
2. Choose an Output Directory.
3. Select a Model from the list picker.
   - On Windows/Linux, models are shown in a categorized tree with a search bar.
   - On macOS, models are listed in an accessible ListBox.
   - Downloaded models appear at the top under "Downloaded Models (Offline)".
4. Select the output format (WAV, FLAC, or MP3 at 320kbps).
5. Click "Start Separation".

4. NATIVE ENSEMBLE (MIXING MODELS)
-------------------------------------------------------------------------------
Tick the "Ensemble" checkbox to select TWO different models. The application 
will process the file with both and fuse the results.
- Ensemble Algorithm: Choose how to mix the tracks (Average, Min, Max, Median).
- Recommended: "avg_wave" (Average) for the most natural sound.

5. CHAINED ENSEMBLE PRESETS & CUSTOM PRESETS
-------------------------------------------------------------------------------
The "Preset" dropdown allows you to run "Chained" processes where the output 
of one model is automatically fed into another.

CREATING CUSTOM PRESETS:
- Click the "Create Preset" button next to the preset dropdown to open the Creator Dialog.
- Set a Name and choose between "Single Model" or "Chained Models".
- Add steps, select models, define which stem to pass, and use the checklist grid to choose which other stems to keep and their custom suffixes.
- Save your preset. It will instantly appear in the preset dropdown.
- To delete, select your custom preset and click the "X" button next to it.

AVAILABLE BUILT-IN PRESETS:
- Vocal Split: Separates Vocals/Inst, then splits Vocals into Lead and Backing.
- Vocal De-Reverb: Separates Vocals, then removes reverb/echo from the voice.
- Ultimate Stems (7 Tracks): High-quality Vocals and Instrumental extraction, 
  then splits Instrumental into Bass, Drums, Piano, Guitar, and Other.
- Ultimate Stems + Drum Split: A 4-pass preset that separates Lead Vocals,
  Backing Vocals, Instrumental, Bass, Piano, Guitar, and splits drums into
  individual components (Kick, Snare, Hi-Hat, Toms, etc., for 10+ tracks total).
- Guitar Extraction: Specialized preset to isolate guitars from the track.
- Male/Female Split: Specifically isolates male vs. female voices.
- Live Restore: Extracts Crowd/Applause, then cleans the remaining audio.
- RVC Vocal Prep: Extracts clean Lead vocals for RVC voice cloning.
- Only Drums: Isolates only the drum kit, removing all other instruments.
- Only Drums + Backing: Extracts the drums (_Drums) and merges the rest of the
  song into one backing track (_No_Drums).

6. OUTPUT OPTIONS AND CLEANUP
-------------------------------------------------------------------------------
- Save in subfolder: Creates a folder named after the song. If disabled, 
  files are saved directly in the output directory.
- Delete silent stems: If enabled, the program deletes generated stems that are 
  empty or very silent (below -50 dB) to keep folders clean.
- Play Stem: Click to play the newly created stems directly in your default 
  system media player.

7. CHUNKING (LONG FILES / OUT-OF-MEMORY PREVENTION)
-------------------------------------------------------------------------------
If you experience crashes with long files or have low VRAM:
- Check "Enable Chunking" and select a chunk size (e.g., 2 min).

8. BATCH PROCESSING (FILES AND FOLDERS)
-------------------------------------------------------------------------------
Process multiple files automatically by dragging them or using the "Browse" 
and "Browse Folder" buttons.

9. TECHNICAL NOTES AND SUFFIXES
-------------------------------------------------------------------------------
Output files will have clear suffixes such as:
_Vocals, _Instrumental, _Lead, _Backing, _DeReverb, _Other, _Extra, _Drums, _No_Drums.
