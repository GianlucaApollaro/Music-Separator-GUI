=====================================================
   Music Separator - macOS (Apple Silicon M1/M2/M3/M4)
=====================================================

Follow these steps to set up and run the application on your Mac.

1. INSTALL PREREQUISITES
-------------------------
The easiest way to install the necessary tools is using Homebrew (https://brew.sh).
Open the Terminal and run:

    brew update
    brew install ffmpeg libsndfile python@3.12

2. FIX FILE PERMISSIONS
-----------------------
macOS often blocks files copied from other computers. You must manually grant execution 
permissions to the script files.

Open the Terminal, type the following command, and press Enter:

    chmod +x "/path/to/this/folder/"*.command

(Tip: You can type 'chmod +x ' and then drag the folder into the terminal window 
to get the path automatically, then add /*.command at the end).

3. INSTALLATION
---------------
Double-click the file: install_mac.command

NOTE: If macOS says the file is "blocked" or "from an unidentified developer":
- Right-click (or Ctrl+Click) the file.
- Select "Open".
- Click the "Open" button in the warning dialog.

This will create a virtual environment (venv_mac) and install all required 
Python libraries, including PyTorch optimized for Apple Silicon (MPS).

4. RUNNING THE APPLICATION
--------------------------
Once the installation is complete, double-click: run_mac.command

(Use the same Right-click -> Open trick for the first time if needed).

5. TROUBLESHOOTING
------------------
- If you get a "ModuleNotFoundError: No module named 'onnxruntime'", 
  open the terminal in this folder and run:
  
  source venv_mac/bin/activate
  pip install onnxruntime

- For best performance, ensure you are using Python 3.12 as installed in Step 1.
