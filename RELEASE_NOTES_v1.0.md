# Release v1.0 🎶

Welcome to the first official release of **Music-Separator-GUI**!

This version provides a complete, easy-to-use interface for separating audio tracks into various stems using advanced AI models.

## New Features and Improvements 🚀

- **Unified Windows & macOS Support**: A single, clean codebase for both platforms.
- **Improved Model List**: Reordered models for better accessibility (especially for screen reader users like NVDA).
- **Interface Fixes**: Eliminated duplicate entries in the model selection dropdown.
- **Multilingual Support**: Fully localized in English, Italian, and Spanish.
- **Ensemble Mode**: Mix and match different models for superior separation results.

## Available Versions 📦

- **Music_Separator_GUI_v1.0_Windows_GPU.7z**: Best for users with a dedicated NVIDIA GPU (requires CUDA). Much faster processing.
- **Music_Separator_GUI_v1.0_Windows_CPU.7z**: For users without a dedicated GPU. Works on any modern PC.

## Installation 🛠️

1. Download the .7z file corresponding to your system.
2. Extract the contents using a tool like [7-Zip](https://www.7-zip.org/) or [WinRAR](https://www.rarlab.com/).
3. Run `Music separator.exe` (or `Music separator CPU.exe`) to start the application.
4. The first time you select a model, the application will automatically download it.
